# iFood Customer Segmentation with K-Means

## Project Overview
This project aims to perform **customer segmentation for iFood** using the **K-Means algorithm** and **unsupervised machine learning techniques**. The goal is to identify customer behavior patterns and group customers with similar characteristics to support business decisions.

The generated clusters can help areas such as **marketing**, **product**, and **customer relationship**, making actions more focused and efficient.

---

## Key Benefits of Segmentation
- Create personalized marketing campaigns for different customer groups  
- Choose the best communication channel for each segment (email, social media, ads, etc.)  
- Identify opportunities to improve existing products or develop new services  
- Improve customer relationships and user experience  
- Support pricing strategies and price tests  
- Focus on the most profitable customers  
- Improve customer service  
- Support **cross-sell** and **upsell** strategies  

---

## Technologies Used
- **Python** – Main programming language  
- **Pandas** – Data manipulation and analysis  
- **NumPy** – Numerical and mathematical operations  
- **Matplotlib** – Data visualization  
- **Seaborn** – Statistical data visualization  
- **Scikit-learn** – Machine Learning library, including the **K-Means** algorithm  

---
## Data Dictionary

| Variable         | Description |
|------------------|------------|
| id_transaction   | Transaction ID. The same ID may contain multiple items within a single order. |
| order_time       | Exact time when the order was placed. |
| location         | Location that processed the order (restaurant unit). |
| item_name        | Item name (pizza, salad, beverage, or dessert). |
| item_quantity   | Quantity of items in the order. |
| latitude         | Latitude of the location where the order was generated. |
| longitude        | Longitude of the location where the order was generated. |
